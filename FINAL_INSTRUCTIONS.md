## 📄 Загрузка MD файлов через Telegram бота

### Как использовать:

1. Откройте Telegram бота
2. Отправьте `/start`
3. Выберите "📰 Статьи" → "➕ Создать"
4. Введите название статьи
5. Введите URL статьи
6. **На шаге "Контент":**
   - Можете ввести текст вручную
   - **ИЛИ отправить .md файл** 📎
7. Бот автоматически прочитает файл и сохранит контент!

### Пример:

```
Шаг 3/4: Введите контент статьи в формате Markdown

💡 Вы можете:
• Ввести текст вручную
• Отправить .md файл
```

Просто прикрепите файл `article.md` и бот сам загрузит весь контент!
