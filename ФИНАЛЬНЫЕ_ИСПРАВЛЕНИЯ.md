## Как тестировать

### Обновление фото:
1. Отзывы → Обновить → введите ID
2. Выберите "📷 Фото"
3. Отправьте фото ИЛИ введите URL
4. Фото сохранится!

### Обновление звезд:
1. Отзывы → Обновить → введите ID
2. Выберите "⭐ Звезды (1-5)"
3. Нажмите на нужное количество звезд
4. Звезды обновятся!

### Пагинация:
1. Отзывы → Список
2. Если больше 7 отзывов, появятся кнопки навигации
3. Нажмите "➡️ Вперед" для следующей страницы
4. Нажмите "⬅️ Назад" для предыдущей
