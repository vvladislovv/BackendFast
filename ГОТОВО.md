## Как протестировать

1. Откройте бота в Telegram
2. Отправьте команду `/start`
3. Выберите "⭐ Отзывы"
4. Нажмите "➕ Создать"

### Вы должны увидеть:

```
➕ Создание отзыва

Шаг 1/5: Выберите имя автора отзыва:

[Иван Петров] [Мария Иванова]
[Алексей Смирнов] [Елена Козлова]
[Дмитрий Соколов] [Анна Морозова]
[✏️ Ввести свое]
```

### Далее:
- Выберите имя → появятся кнопки компаний
- Выберите компанию → появятся шаблоны отзывов
- Выберите шаблон → появятся кнопки звезд
- Выберите звезды → появятся кнопки фото

## Редактирование

1. Выберите "⭐ Отзывы" → "✏️ Обновить"
2. Введите ID отзыва (например: 1)
3. Появятся кнопки выбора поля
4. При выборе "⭐ Звезды" появятся кнопки 1-5

## Логи бота

## Если что-то не работает

Перезапустите бот еще раз:

