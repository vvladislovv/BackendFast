# Тестовая статья о разработке

Это **тестовая статья** для проверки загрузки Markdown файлов через Telegram бота.

## Основные возможности

- Загрузка MD файлов напрямую
- Автоматическое сохранение форматирования
- Поддержка всех Markdown элементов

### Пример кода

```python
def hello_world():
    print("Hello, HackTaika!")
```
