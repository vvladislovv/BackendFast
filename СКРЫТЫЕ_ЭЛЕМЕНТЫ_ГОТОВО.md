## Как использовать

### Просмотр скрытых элементов:
1. Выберите категорию (Отзывы, Вакансии, и т.д.)
2. Нажмите "🔒 Скрытые"
3. Увидите список всех скрытых элементов

### Быстрое скрытие/показ:
1. Найдите элемент по ID через "🔍 Найти по ID"
2. Используйте кнопки быстрого переключения
3. Или используйте "👁️ Скрыть/Показать" для ввода ID

### Отображение в списках:
- **Обычный список:** показывает только видимые элементы
- **Скрытые:** показывает только скрытые элементы со значком 🔒
- **Рейтинг и звезды:** отображаются вместе для удобства

## Примеры отображения
